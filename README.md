# 🧩 Sudoku Showdown

**Sudoku Showdown** is a real-time multiplayer web app where players can join or create rooms and collaboratively or competitively solve Sudoku puzzles. Built with **React**, **Express**, and **WebSockets**, it offers a fast-paced and interactive way to enjoy Sudoku with friends or strangers.

## 🚀 Features

- 🔧 Create or join public/private Sudoku rooms
- 👥 Play cooperatively or compete to solve the same puzzle
- 🔄 Real-time updates via WebSockets
- 🧠 Fair puzzle generation (same grid for all players in a room)
- 🎯 Track player progress, input accuracy, and time to solve
- 🏆 Display winner (for competitive mode)

## 🛠️ Tech Stack

- **Frontend**: React, Tailwind CSS
- **Backend**: Node.js, Express
- **Real-Time**: WebSockets (using `socket.io`)
- **Sudoku Generator**: Custom algorithm or third-party library
- **State Management**: React Context or Redux (optional)
